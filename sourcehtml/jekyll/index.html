Discover Aztec Diamonds

The goal of this website is to offer an easy-to-use tool for generating random tilings of shapes in the plane, as well as counting the number of such tilings.

Follow the link below to get started, generate and count tilings.

Generate tiling

The software is available online, within your browser, but a fully optimized desktop application is also available.

Download

By clicking on download, you will be redirected to the GitHub website, where Windows, Mac and Linux version of the software are available for download (Java required).

The code of the project is available on the GitHub website. Any suggestion or comments are most welcome!
